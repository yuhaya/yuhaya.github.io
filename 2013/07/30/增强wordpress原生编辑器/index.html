增强wordpress原生编辑器 

//增强编辑器开始
function add_editor_buttons($buttons) {
    $buttons[] = 'fontselect';
    $buttons[] = 'fontsizeselect';
    $buttons[] = 'cleanup';
    $buttons[] = 'styleselect';
    $buttons[] = 'hr';
    $buttons[] = 'del';
    $buttons[] = 'sub';
    $buttons[] = 'sup';
    $buttons[] = 'copy';
    $buttons[] = 'paste';
    $buttons[] = 'cut';
    $buttons[] = 'undo';
    $buttons[] = 'image';
    $buttons[] = 'anchor';
    $buttons[] = 'backcolor';
    $buttons[] = 'wp_page';
    $buttons[] = 'charmap';
    return $buttons;
}
add_filter("mce_buttons_3", "add_editor_buttons");
//增强编辑器结束

    **wordpress编辑器按钮调用的Key：**

    剪切(cut)复制(copy)粘贴(paste)撤销(undo)重做(redo)居中(justifycenter)&nbsp;

    加粗(bold)斜体(italic)左对齐(justifyleft)右对齐(justfyright)&nbsp;

    两端对齐(justfyfull)插入超链接(link)取消超链接(unlink)插入图片(image)&nbsp;

    清除格式(removeformat)下划线(underline)删除线(strikethrough)

    锚文本(anchor)新建文本(newdocument) 字体颜色(forecolor)字体背景色(backcolor)&nbsp;

    格式选择(formmatselect)字体选择(fontselect)字号选择(fontsizeselect)&nbsp;

    样式选择(styleselect)无序列表(bullist)编号列表(numlist) 减少缩进(outdent)

    缩进(indent)帮助(wp_help) 打开HTML代码编辑器(code)水平线(hr)

    清除冗余代码(cleanup) 上标(sub)下标(sup)特殊符号(charmap)&nbsp;

    插入more标签(wp_more)插入分页标签(wp_page) 隐藏按钮显示开关(wp_adv)&nbsp;

    隐藏按钮区起始部分(wp_adv_start) 隐藏按钮区结束部分(wp_adv_end) 拼写检查(spellchecker)
