Android开发环境的安装

Android开发环境的安装

<span style="font-family:Microsoft YaHei;">1 IDE</span>

<span style="font-family:Microsoft YaHei;">Android可以使用开发的IDE有Eclipse 或者 Android Studio。Android Studio还处于v 0.1.x版本，是early access preview。所以我还是选择Eclipse + ADT的方法开发。</span>

<span style="font-family:Microsoft YaHei;">其实没有什么复杂，官网已经为开发者做了一个开发环境集成包（ADT Bundle）。ADT是Android Develop Tools。这个集成包包含了eclipse，SDK，SDK Manager。</span>

[<span style="font-family:Microsoft YaHei;">http://developer.android.com/sdk/index.html</span>](http://developer.android.com/sdk/index.html)

<span style="font-family:Microsoft YaHei;">具体使用只需要下载Android ADT Bundle的zip，解压，然后启动eclipse就可以了。</span>


<span style="font-family:Microsoft YaHei;">2 安装不同版本的sdk</span>

<span style="font-family:Microsoft YaHei;">Bundle默认是安装了最新版本的SDK（4.2.2）。当然现在市面上的各种Android版本都存在，我们需要安装不同版本的SDK来开发。SDK Manager就是管理SDK的工具。</span>

<span style="font-family:Microsoft YaHei;">在bundle下运行SDK Manager</span>

[![Image](http://images.cnitblog.com/blog/136188/201307/08101433-b1cb1059df4e4fc2abe54ecae04e8cec.png "Image")](http://images.cnitblog.com/blog/136188/201307/08101432-9229801a85f94a12ac27949eb374a1f6.png)

<span style="font-family:Microsoft YaHei;">勾选对应版本的SDK，从这里基本可以知道一个Android版本对应着一个版本的API。</span>

<span style="font-family:Microsoft YaHei;">其中每个包都有这么几个文件：</span>


<span style="font-family:Microsoft YaHei;">Documentation for Android SDK：安卓SDK的文档</span>


<span style="font-family:Microsoft YaHei;">SDK Platform：Android的SDK套件，必须要安装的</span>


<span style="font-family:Microsoft YaHei;">Samples for SDK：SDK的使用例子，在安装后的adt-bundle-XXXXsdksamples 目录中会看到对应文件</span>


<span style="font-family:Microsoft YaHei;">Sources for Android SDK：SDK的源码，在安装后的adt-bundle-windows-x86-20130522sdksources 目录中会看到对应文件</span>


<span style="font-family:Microsoft YaHei;">Google APIs：谷歌地图提供的一套API，如果你的应用不使用谷歌地图，是可以不用装的，但是建议还是要装</span>


<span style="font-family:Microsoft YaHei;">ARM EABI v7a System Image</span>


<span style="font-family:Microsoft YaHei;">Intel x86 Atom System Image</span>


<span style="font-family:Microsoft YaHei;">MIPS System Image</span>

<span style="font-family:Microsoft YaHei;">这三个是给模拟器用的，模拟器在不同CPU平台下的系统印象文件。必须安装，否则模拟器打不开。</span>

<span style="font-family:Microsoft YaHei;">在安装后的adt-bundle-windows-x86-20130522sdksystem-images目录中会看到对应文件</span>


<span style="font-family:Microsoft YaHei;">3 安装Eclipse的ADT插件</span>

<span style="font-family:Microsoft YaHei;">官方的Bundle上已经有了一些ADT插件，如果没有的话，我们可以按照：</span>

[<span style="font-family:Microsoft YaHei;">http://developer.android.com/sdk/installing/installing-adt.html</span>](http://developer.android.com/sdk/installing/installing-adt.html)

<span style="font-family:Microsoft YaHei;">安装ADT插件。</span>


<span style="font-family:Microsoft YaHei;">4 创建模拟器</span>

<span style="font-family:Microsoft YaHei;">在Eclipse中的Android Virtual Devices中New一个虚拟机（AVD）</span>

[![Image(1)](http://images.cnitblog.com/blog/136188/201307/08101434-b41f038ba6f3414693a5635d4a3cd0ef.png "Image(1)")](http://images.cnitblog.com/blog/136188/201307/08101434-a40b01e719ce44609e7bb9a94b3bc36b.png)

**<span style="font-family:Microsoft YaHei;">AVD Name</span>**<span style="font-family:Microsoft YaHei;">：模拟器的名字</span>

**<span style="font-family:Microsoft YaHei;">Device</span>**<span style="font-family:Microsoft YaHei;">：设备名字，包括屏幕尺寸的设置</span>

**<span style="font-family:Microsoft YaHei;">Target</span>**<span style="font-family:Microsoft YaHei;">：要运行的Android版本</span>

**<span style="font-family:Microsoft YaHei;">CPU/ABI</span>**<span style="font-family:Microsoft YaHei;">：模拟器要运行的系统印象文件</span>

**<span style="font-family:Microsoft YaHei;">Keyboard</span>**<span style="font-family:Microsoft YaHei;">：是否支持外部键盘</span>

**<span style="font-family:Microsoft YaHei;">Skin</span>**<span style="font-family:Microsoft YaHei;">：模拟器外观</span>

**<span style="font-family:Microsoft YaHei;">Front Camera</span>**

**<span style="font-family:Microsoft YaHei;">Back Camera</span>**

<span style="font-family:Microsoft YaHei;">前置和后置摄像头</span>

**<span style="font-family:Microsoft YaHei;">Memory Options</span>**<span style="font-family:Microsoft YaHei;">：</span>

**<span style="font-family:Microsoft YaHei;">RAM</span>**<span style="font-family:Microsoft YaHei;">：模拟器的内存空间</span>

**<span style="font-family:Microsoft YaHei;">VM Heap</span>**<span style="font-family:Microsoft YaHei;">：Android系统是运行在Dalvik虚拟机上的，VM Heap就是指的虚拟机最大占用内存，也就是单个应用的最大占用内存。</span>

**<span style="font-family:Microsoft YaHei;">Internal Storage</span>**<span style="font-family:Microsoft YaHei;">：内部存储大小。内部存储就是将数据保存在设备的内部存储器中。</span>

**<span style="font-family:Microsoft YaHei;">SD Card</span>**<span style="font-family:Microsoft YaHei;">：SD 卡的大小。</span>


<span style="font-family:Microsoft YaHei;">5 创建HelloWorld</span>

<span style="font-family:Microsoft YaHei;">Eclipse中New--Andriod Project -- 创建个HelloWorld项目。</span>

<span style="font-family:Microsoft YaHei;">这步基本都是使用默认选项</span>


<span style="font-family:Microsoft YaHei;">6 HelloWorld项目挂载AVD</span>

<span style="font-family:Microsoft YaHei;">Eclipse -- Run -- Run Configurations</span>

<span style="font-family:Microsoft YaHei;">右键Android Application -- New</span>

<span style="font-family:Microsoft YaHei;">Project中选择HelloWorld项目 -- Target中选择刚刚创建的AVD -- Apply</span>

<span style="font-family:Microsoft YaHei;">这样，运行项目就自动开启了虚拟机运行</span>

[![Image(2)](http://images.cnitblog.com/blog/136188/201307/08101435-9c9b8a22b8fc42bfa9b029ba6b71781d.png "Image(2)")](http://images.cnitblog.com/blog/136188/201307/08101435-5ac805b2a9be4e88a5dd5b0ddd292a72.png)
