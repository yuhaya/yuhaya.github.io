JS中的prototype 

<span style="font-size:13px;">

    **<span style="font-family:Microsoft YaHei;">JS中的phototype是JS中比较难理解的一个部分</span>**

    **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

    **<span style="font-family:Microsoft YaHei;">本文基于下面几个知识点:</span>**

    **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

    <span style="line-height:1.8;font-size:14pt;font-family:'Microsoft YaHei';">**1 原型法设计模式**</span>

    **<span style="font-family:'Microsoft YaHei';font-size:12px;">在.Net中可以使用clone()来实现原型法</span>**

    **<span style="font-family:'Microsoft YaHei';font-size:12px;">原型法的主要思想是，现在有1个类A,我想要创建一个类B,这个类是以A为原型的,并且能进行扩展。我们称B的原型为A。</span>**

    **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

    <span style="line-height:1.8;font-size:14pt;font-family:'Microsoft YaHei';">**2 javascript的方法可以分为三类：**</span>

    **<span style="font-family:Microsoft YaHei;">a 类方法</span>**

    **<span style="font-family:Microsoft YaHei;">b 对象方法</span>**

    **<span style="font-family:Microsoft YaHei;">c 原型方法</span>**

    **<span style="font-family:Microsoft YaHei;">例子：</span>**

                <span style="line-height:1.5;color:#0000FF;font-family:'Microsoft YaHei';">**function**</span><span style="line-height:1.5;color:#000000;">**<span style="font-family:Microsoft YaHei;">&nbsp;People(name)</span>**

{

this.name=name;

//对象方法

this.Introduce=function(){

alert("My name is "+this.name);

//类方法

People.Run=function(){

People.prototype.IntroduceChinese=function(){

            <span style="line-height:1.8;font-size:14pt;font-family:'Microsoft YaHei';">**3 obj1.func.call(obj)方法**</span>

            **<span style="font-family:Microsoft YaHei;">意思是将obj看成obj1,调用func方法</span>**

            **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

            **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

            **<span style="font-family:Microsoft YaHei;">好了，下面一个一个问题解决：</span>**

            **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

            <span style="line-height:1.8;font-size:14pt;font-family:'Microsoft YaHei';">**prototype是什么含义？**</span>

            **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

            **<span style="font-family:Microsoft YaHei;">javascript中的每个对象都有prototype属性，Javascript中对象的prototype属性的解释是：返回对象类型原型的引用。</span>**

            **<span style="font-family:Microsoft YaHei;">A.prototype = new B();</span>**

            **<span style="font-family:Microsoft YaHei;">理解prototype不应把它和继承混淆。A的prototype为B的一个实例，可以理解A将B中的方法和属性全部克隆了一遍。A能使用B的方法和属性。这里强调的是克隆而不是继承。可以出现这种情况：A的prototype是B的实例，同时B的prototype也是A的实例。</span>**

            **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

            **<span style="font-family:Microsoft YaHei;">先看一个实验的例子：</span>**

        <div class="cnblogs_code" style="background-color:#F5F5F5;font-family:'Courier New';font-size:12px;border:1px solid #CCCCCC;padding:5px;margin:5px 0px;">
            <div class="cnblogs_code_toolbar">

            </div>
            <div>
                <span style="line-height:1.5;color:#0000FF;font-family:'Microsoft YaHei';">**function**</span><span style="line-height:1.5;color:#000000;">**<span style="font-family:Microsoft YaHei;">&nbsp;baseClass()</span>**

{

this.showMsg = function()

{

     alert("baseClass::showMsg");   

}

}

function extendClass()

{

}

extendClass.prototype = new baseClass();

var instance = new extendClass();

instance.showMsg(); // 显示baseClass::showMsg 

        **<span style="font-family:Microsoft YaHei;">我们首先定义了baseClass类，然后我们要定义extentClass，但是我们打算以baseClass的一个实例为原型，来克隆的extendClass也同时包含showMsg这个对象方法。</span>**

        **<span style="font-family:Microsoft YaHei;">extendClass.prototype = new baseClass()就可以阅读为：extendClass是以baseClass的一个实例为原型克隆创建的。</span>**

        **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

        **<span style="font-family:Microsoft YaHei;">那么就会有一个问题，</span>**<span style="line-height:1.8;font-size:14pt;font-family:'Microsoft YaHei';">**如果extendClass中本身包含有一个与baseClass的方法同名的方法会怎么样？**</span>

        **<span style="font-family:Microsoft YaHei;">下面是扩展实验2：</span>**

    <div class="cnblogs_code" style="background-color:#F5F5F5;font-family:'Courier New';font-size:12px;border:1px solid #CCCCCC;padding:5px;margin:5px 0px;">
        <div class="cnblogs_code_toolbar">

            <span style="line-height:1.5;color:#0000FF;font-family:'Microsoft YaHei';">**function**</span><span style="line-height:1.5;color:#000000;">**<span style="font-family:Microsoft YaHei;">&nbsp;baseClass()</span>**

{

    this.showMsg = function()

    {

        alert("baseClass::showMsg");   

    }

}

function extendClass()

{

    this.showMsg =function ()

    {

        alert("extendClass::showMsg");

    }

}

extendClass.prototype = new baseClass();

var instance = new extendClass();

instance.showMsg();//显示extendClass::showMsg 

        **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

        **<span style="font-family:Microsoft YaHei;">实验证明：函数运行时会先去本体的函数中去找，如果找到则运行，找不到则去prototype中寻找函数。或者可以理解为prototype不会克隆同名函数。</span>**

        **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

        **<span style="font-family:Microsoft YaHei;">那么又会有一个新的问题：</span>**

        <span style="line-height:1.8;font-size:14pt;font-family:'Microsoft YaHei';">**如果我想使用extendClass的一个实例instance调用baseClass的对象方法showMsg怎么办？**</span>

        **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

        **<span style="font-family:Microsoft YaHei;">答案是可以使用call:</span>**

    <div class="cnblogs_code" style="background-color:#F5F5F5;font-family:'Courier New';font-size:12px;border:1px solid #CCCCCC;padding:5px;margin:5px 0px;">
        <span style="line-height:1.5;color:#000000;font-family:'Microsoft YaHei';">**extendClass.prototype&nbsp;**</span><span style="line-height:1.5;color:#000000;font-family:'Microsoft YaHei';">**=**</span><span style="line-height:1.5;color:#000000;font-family:'Microsoft YaHei';">**&nbsp;**</span><span style="line-height:1.5;color:#0000FF;font-family:'Microsoft YaHei';">**new**</span><span style="line-height:1.5;color:#000000;">**<span style="font-family:Microsoft YaHei;">&nbsp;baseClass();</span>**

var instance = new extendClass();

var baseinstance = new baseClass();

baseinstance.showMsg.call(instance);//显示baseClass::showMsg 

        **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

        **<span style="font-family:Microsoft YaHei;">这里的baseinstance.showMsg.call(instance);阅读为“将instance当做baseinstance来调用，调用它的对象方法showMsg”</span>**

        **<span style="font-family:Microsoft YaHei;">好了，这里可能有人会问，为什么不用baseClass.showMsg.call(instance);</span>**

        **<span style="font-family:Microsoft YaHei;">这就是对象方法和类方法的区别，我们想调用的是baseClass的对象方法</span>**

        **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

        <span style="line-height:1.8;font-size:14pt;font-family:'Microsoft YaHei';">**最后，下面这个代码如果理解清晰，那么这篇文章说的就已经理解了：**</span>

        **<span style="font-family:Microsoft YaHei;">&nbsp;</span>**

    <div class="cnblogs_code" style="background-color:#F5F5F5;font-family:'Courier New';font-size:12px;border:1px solid #CCCCCC;padding:5px;margin:5px 0px;">
        <div class="cnblogs_code_toolbar">

            <span style="line-height:1.5;color:#000000;font-family:'Microsoft YaHei';">**&lt;**</span><span style="line-height:1.5;color:#000000;font-family:'Microsoft YaHei';">**script&nbsp;type**</span><span style="line-height:1.5;color:#000000;font-family:'Microsoft YaHei';">**=**</span><span style="line-height:1.5;color:#000000;font-family:'Microsoft YaHei';">**"**</span><span style="line-height:1.5;color:#000000;font-family:'Microsoft YaHei';">**text/javascript**</span><span style="line-height:1.5;color:#000000;font-family:'Microsoft YaHei';">**"**</span><span style="line-height:1.5;color:#000000;font-family:'Microsoft YaHei';">**&gt;**</span><span style="line-height:1.5;color:#000000;">

function baseClass()

{

    this.showMsg = function()

    {

        alert("baseClass::showMsg");   

    }

   

    this.baseShowMsg = function()

    {

        alert("baseClass::baseShowMsg");

    }

}

baseClass.showMsg = function()

{

    alert("baseClass::showMsg static");

}

function extendClass()

{

    this.showMsg =function ()

    {

        alert("extendClass::showMsg");

    }

}

extendClass.showMsg = function()

{

    alert("extendClass::showMsg static")

}

extendClass.prototype = new baseClass();

var instance = new extendClass();

instance.showMsg(); //显示extendClass::showMsg

instance.baseShowMsg(); //显示baseClass::baseShowMsg

instance.showMsg(); //显示extendClass::showMsg

baseClass.showMsg.call(instance);//显示baseClass::showMsg static

var baseinstance = new baseClass();

baseinstance.showMsg.call(instance);//显示baseClass::showMsg

</script>
