CSS实现半透明背景 兼容IE6

透明度为90%的纯色背景，代码如下：

[code lang="css"]

header{

filter:progid:DXImageTransform.Microsoft.gradient(enabled='true', startColorstr='#E518515F', endColorstr='#E518515F'); background:rgba(24, 81, 95, 0.9); } :root #header { filter:none; } [/code]

其中filter是IE的滤镜，background:rgba是CSS3属性，最后一句是针对IE9的。所以这段CSS是兼容所有浏览器的，可放心使用…

扩展：

用css透明属性，这里跟上面的方法比较是分不同场合使用不同方法的。有的时候我们的需求并不是一个纯颜色的半透明，我们可能会用到图片半透明，而第一种方法是做不到的。所以我们需要用这种方法。

[code lang="css"]opacity：0.88；filter:alpha(opacity=88)[/code]

具体自己可以去试着写一写。这种方法的有一个弊端就是，设置了透明的div（或其他）里面的内容或者说子集都有继承它的透明属性，也会跟着透明，而多数时候我们是不需要这样的。

两个同级的div:

[code lang="html"] <div class="box1">图上</div> <div class="box2">内容</div> [/code]

解决：把图片(或设背景)放在box1里面，并设置半透明；把实际内容放在box2里面，并把box2定位在box1之上，就可以实现了。
