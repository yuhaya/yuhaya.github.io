zf2输出json格式的数据

或许你认为，输出json有什么麻烦的，一个json_encode()函数不就搞定了！但是在zf2中，输出json确实不是这么简单地事，仅仅简单的使用json_encode()的话，你无法禁用loyout，然后你的json数据就会乱掉，好了，不说废话了，介绍下在zf2中怎么输出json数据。

使用View Helper – JSON输出json数据，代码如下，首先是配置view_manager，加入如下配置

'strategies'=>array(
'ViewJsonStrategy',
),
配置完成如下 
'view_manager'=>array(
'strategies'=>array(
'ViewJsonStrategy',
),
),
然后修改你的zf2.local/module/Application/src/Application/Controller/IndexController.php的indexAction方法，如下 
publicfunctionindexAction()
{
$data=array('k1'=> 1,'k2'=> 2);
returnnewJsonModel($data);
}
然后访问这个方法，你就可以看到输出了。 在zf2还有一种方法也可以正确的输出json,直接修改zf2.local/module/Application/src/Application/Controller/IndexController.php的indexAction方法，如下 
publicfunctionindexAction()
{
$data=array('k1'=> 1,'k2'=> 2);
$response=$this->getResponse();
$response->setContent(ZendJsonJson::encode(array(

'data'=>$data
)));

return$response;
}
                        </div>
                    </div>
                </td>
            </tr>
        </tbody>
    </table>
</div>


然后访问这个方法，你也可以看到输出。

这两种输出有什么区别呐？他们都会自动禁用layout，第一种方式，它会输出application/json的header type,第二中则是普通text文本，推荐使用第一种。
