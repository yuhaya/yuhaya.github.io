CentOS配置本地yum源（使用镜像iso文件）

本人在使用yum安装软件的时候，感觉最不爽的是网络不佳时，安装的速度特别慢。所以，个人就上网search了一下如何使用Linux的安装文件作为其yum源。经过几次尝试，已经可以成功的配置了。下面是详细的介绍：

（1）个人的使用环境

本人的CentOS是安装在vmware上的：VMware workstation 7.1.1 + CentOS 5.5

（2）yum使用的相关文件介绍

&nbsp;&nbsp; &nbsp;CentOS使用的yum源是在/etc/yum.repos.d/目录下的两个文件中进行配置的CentOS-Base.repo和CentOS-Media.repo，其中，CentOS-Base.repo是配置网络yum源，CentOS-Media.repo是配置本地yum源。

&nbsp;

[root@localhost ~]# cd /etc/yum.repos.d/

[root@localhost yum.repos.d]# ll

total 16

-rw-r--r-- 1 root root 2245 Apr 26 &nbsp;2010 CentOS-Base.repo

-rw-r--r-- 1 root root &nbsp;626 Apr 26 &nbsp;2010 CentOS-Media.repo

我们可以看看CentOS-Media.repo中的内容：

[root@localhost yum.repos.d]# more CentOS-Media.repo&nbsp;

# CentOS-Media.repo

#

# This repo is used to mount the default locations for a CDROM / DVD on

# &nbsp;CentOS-5\. &nbsp;You can use this repo and yum to install items directly off the

# &nbsp;DVD ISO that we release.

#

# To use this repo, put in your DVD and use it with the other repos too:

# &nbsp;yum --enablerepo=c5-media [command]

# &nbsp;

# or for ONLY the media repo, do this:

#

# &nbsp;yum --disablerepo=/* --enablerepo=c5-media [command]

&nbsp;

[c5-media]

name=CentOS-$releasever - Media

baseurl=file:///media/CentOS/

&nbsp;&nbsp; &nbsp; &nbsp; &nbsp;file:///media/cdrom/

&nbsp;&nbsp; &nbsp; &nbsp; &nbsp;file:///media/cdrecorder/

gpgcheck=1

enabled=0

gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-CentOS-5

&nbsp;&nbsp; &nbsp;其中，baseurl是本地yum源的位置，当然，根据我们iso的配置位置，这里是可以进行修改的。不过，我只是将iso文件挂载到/media/CentOS/目录，所以，我也没有做修改。
（3）挂载iso文件
    本人的iso文件是在宿主机上，所以，通过vmware的文件共享，将宿主机上的iso文件挂载到/mnt/hgfs/目录下：
VM->Settings->Options选项卡->Shared Folders->Set Folder sharing as 'Always enabled' 
&nbsp;&nbsp; &nbsp;使用Add button选取本地iso文件的目录，我的CentOS放在D:/Software/os/目录下，所以，添加后的截图如下：

[![0_1298959053xKeH](http://www.3wmao.com/wp-content/uploads/2014/01/0_1298959053xKeH.jpg)](http://www.3wmao.com/wp-content/uploads/2014/01/0_1298959053xKeH.jpg)

<div>
    点击'OK'，完成共享。
</div>
<div>
    &nbsp;&nbsp; &nbsp;在CentOS中，我们可以在/mnt/hgfs/目录下发现一个os/子目录，CentOS的iso文件也就在该目录下，本人共享以后，iso文件在CentOS的路径为：/mnt/hgfs/os/CentOS-5.5-i386-bin-DVD.iso
</div>
<div>
</div>
<div>
    将该镜像文件挂载到/media/CentOS/
</div>
<div>
    <div>
        [root@localhost yum.repos.d]# mount -o loop /mnt/hgfs/os/CentOS-5.5-i386-bin-DVD.iso /media/CentOS/
    </div>
    <div>
    </div>
    <div>
        （4）使用本地yum源安装软件
    </div>
    <div>
        使用本地yum源的方法在CentOS-Media.repo中已经描述了：
    </div>
    <div>
        （4.1）除了使用本地yum源，也会是使用其他的yum源
    </div>
    <div>
        yum --enablerepo=c5-media [command]
    </div>
    <div>
        （4.2）只是用本地yum源
    </div>
    <div>
        yum --disablerepo=/* --enablerepo=c5-media [command]
    </div>
</div>
<div>
</div>
<div>
    下面是用本地yum源安装mysql
</div>
<div>
    <div>
        [root@localhost yum.repos.d]# yum --disablerepo=/* --enablerepo=c5-media install mysql-server
