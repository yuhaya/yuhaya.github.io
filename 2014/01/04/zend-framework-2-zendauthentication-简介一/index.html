ZEND FRAMEWORK 2 : Zend\Authentication 之 AuthenticationService 

zf2的认证分为主要的两部分：
  •     认证适配器

  •     认证信息的存贮

    一下是部分认证服务的代码

//file:Zend/Authentication/AuthenticationService.php

 /**
     * Persistent storage handler
     *
     * @var Storage\StorageInterface
     */
    protected $storage = null;

    /**
     * Authentication adapter
     *
     * @var Adapter\AdapterInterface
     */
    protected $adapter = null;

    /**
     * Constructor
     *
     * @param  Storage\StorageInterface $storage
     * @param  Adapter\AdapterInterface $adapter
     */
    public function __construct(Storage\StorageInterface $storage = null, Adapter\AdapterInterface $adapter = null)
    {
        if (null !== $storage) {
            $this->setStorage($storage);
        }
        if (null !== $adapter) {
            $this->setAdapter($adapter);
        }
    }
从上面可以看出认证服务主要由适配器对象以及存储对象来组成 而验证： 
 /**
     * Authenticates against the supplied adapter
     *
     * @param  Adapter\AdapterInterface $adapter
     * @return Result
     * @throws Exception\RuntimeException
     */
    public function authenticate(Adapter\AdapterInterface $adapter = null)
    {
        if (!$adapter) {
            if (!$adapter = $this->getAdapter()) {
                throw new Exception\RuntimeException('An adapter must be set or passed prior to calling authenticate()');
            }
        }
        $result = $adapter->authenticate();

        /**
         * ZF-7546 - prevent multiple successive calls from storing inconsistent results
         * Ensure storage has clean state
         */
        if ($this->hasIdentity()) {
            $this->clearIdentity();
        }

        if ($result->isValid()) {
            $this->getStorage()->write($result->getIdentity());
        }

        return $result;
    }
认证服务的验证实质上是调用了认证适配器对象的&nbsp;authenticate 方法，然后在调用认证存储服务存储识别信息

<span>官网说明了两中认证方式</span>
  •     直接调用 认证适配器对象的 authenticate  方法

  •     调用认证服务的 authenticate  的方法

    而这两个方式的区别就是，通过认证服务的方式会默认的在验证成功之后将身份识别信息存储

    上图中adapter目录以及storage目录下主要存放认证数据适配器接口文件以及认证存储适配器文件，其他的则为zf2默认向用户提供的对应常用的实现该接口的类。

    再需要说明一点的就是zf2默认认证存储方式为session
