Cе’ҢC++дёӯConst дҝ®йҘ°жҢҮй’Ҳ

CгҖҒC++дёӯConstдҝ®йҘ°жҢҮй’Ҳж—¶жҖ»жҳҜи®°дёҚдҪҸеҲ°еә•дҪңз”Ёзҡ„жҳҜжҢҮй’ҲиҝҳжҳҜжҢҮй’ҲжҢҮеҗ‘зҡ„еҸҳйҮҸеҶ…е®№пјҡ

жңҖиҝ‘жҖ»з»“дәҶдёҖдёӘз»қжӢӣпјҢеҸҜд»Ҙз»қеҜ№зҗҶжё…жҘҡпјҒйӮЈе°ұжҳҜе°ұиҝ‘еҺҹеҲҷ

жүҖи°“е°ұиҝ‘еҺҹеҲҷпјҢж„ҸжҖқе°ұжҳҜconstйқ е“ӘдёӘиҝ‘пјҢе°ұдҪңз”ЁеңЁйӮЈдёӘдёҠйқў.

constйқ intзұ»еһӢе…ій”®еӯ—иҝ‘пјҢйӮЈе°ұжҳҜиҜҙдҝ®йҘ°зҡ„жҳҜж•ҙеҪўеҸҳйҮҸпјӣconstйқ жҢҮй’ҲеҸҳйҮҸеҗҚиҝ‘пјҢйӮЈе°ұжҳҜиҜҙдҝ®йҘ°иҜҘжҢҮй’ҲпјҢиҜҙжҳҺе®ғдёҚжҳҜжҢҮй’ҲеҸҳйҮҸпјҢиҖҢжҳҜжҢҮй’ҲеёёйҮҸпјӣ;

дҪҶжҳҜжңүдёҖдёӘй—®йўҳпјҡж—ўз„¶иҜҙеҲ°жҳҜconstдҝ®йҘ°жҢҮй’ҲпјҢйӮЈе°ұдёҖе®ҡжңүеҰӮ char const * pContent; йӮЈиҝҷдёӘconstеңЁ*е’ҢеҸҳйҮҸеҗҚд№Ӣй—ҙпјҢеҲ°еә•йқ и°Ғиҝ‘е‘ўпјҹ

иҝҷйҮҢиҰҒи§ЈйҮҠиҜҙжҳҺзҡ„жҳҜпјҡconstйқ и°Ғиҝ‘жҢҮзҡ„жҳҜзұ»еһӢпјҲеҰӮintгҖҒcharзӯүпјүе’ҢеҸҳйҮҸеҗҚпјҲеҰӮиҝҷйҮҢзҡ„pContentпјүпјҢдёҚиғҪз®—*пјҢеӣ жӯӨиҝҷйҮҢconstеҪ“з„¶йқ charиҝ‘пјҢжүҖд»ҘpContentжҳҜдёҖдёӘжҢҮеҗ‘еӯ—з¬ҰеёёйҮҸзҡ„жҢҮй’Ҳ;

жңүдәҶиҝҷдёҖжқЎе°ұиҝ‘еҺҹеҲҷпјҢconstй—®йўҳиҝҺеҲғиҖҢи§ЈпјҒ;

1) constеңЁеүҚйқў;

2) constеңЁеҗҺйқўпјҢдёҺдёҠйқўзҡ„еЈ°жҳҺеҜ№зӯү

=================================================================================;

BjarneеңЁд»–зҡ„The C++ Programming LanguageйҮҢйқўз»ҷеҮәиҝҮдёҖдёӘеҠ©и®°зҡ„ж–№жі•пјҡ жҠҠдёҖдёӘеЈ°жҳҺд»ҺеҸіеҗ‘е·ҰиҜ»гҖӮ;

char const cp;;( иҜ»жҲҗ pointer to );

cp is a const pointer to char пјҡcpжҳҜдёҖдёӘжҢҮеҗ‘еӯ—з¬ҰеһӢзҡ„еёёйҮҸжҢҮй’ҲпјҢжҢҮй’ҲдёҚиғҪж”№еҸҳпјӣ;

const char * p;;

p is a pointer to const charпјҡcpжҳҜдёҖдёӘжҢҮеҗ‘еӯ—з¬ҰеһӢеёёйҮҸзҡ„жҢҮй’ҲпјҢжҢҮй’ҲжҢҮеҗ‘зҡ„еҖјдёҚиғҪйҡҸдҫҝж”№еҸҳпјӣ;

char const * p; еҗҢдёҠгҖӮ;
